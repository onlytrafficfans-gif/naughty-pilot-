# Content Library

Organize, tag, and repurpose your media so nothing shoots twice.

## Folder spine
/raw (untouched shoots) · /edited (export-ready) · /published (with date + platform) · /retired

## Tagging every asset
- Angle: playful / luxury / girl-next-door / fitness
- Format: 1:1 / 9:16 / 16:9
- Performance: CTR band after first run (A/B/C)
- Rights: solo / collab (collab = check release before reuse)

## Repurposing rules
- One 3-hour shoot = 12+ feed assets + 20+ story frames
- A-band assets re-enter rotation after 60 days rest
- C-band assets never re-run — recrop or retire

## The rule
If you can't find it in 30 seconds, you don't own it — the library does.
