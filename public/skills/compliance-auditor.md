# Compliance Auditor

Flag policy violations before you spend — not after the ban.

## Pre-flight checklist (every creative, every network)
- [ ] 21+ age gate on every placement
- [ ] No platform trademarks in ad copy or creative
- [ ] Creative passes the *strictest* target network's rules (audit once, run anywhere)
- [ ] No health/income/relationship claims
- [ ] Landing page matches ad promise (mismatch = account flag)
- [ ] 2257 records current for every person appearing in creative
- [ ] Collab content: signed release on file before it enters rotation

## Red lines (never)
- Implying user-generated/candid content that isn't
- Geo-targeting jurisdictions where the content category is restricted
- Recycling a rejected creative to a different network unchanged

## Cadence
Full audit monthly; spot-audit every new creative before its first dollar.
