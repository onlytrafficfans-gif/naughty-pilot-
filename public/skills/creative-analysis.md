# Creative Analysis

Score and rank photos by predicted click-through before spending on them.

## Scoring rubric (0–10 each)
- Face visibility & eye contact: face-forward with direct gaze consistently outperforms
- Lighting: natural > softbox > flash; avoid mixed color temperature
- Composition: single clear subject, minimal background clutter
- Angle match: does the shot match the campaign angle (playful / luxury / girl-next-door)?
- Policy safety: would this pass the strictest network's review? If unsure, it fails.

## Ranking process
1. Score every candidate on the 5 axes; sum to /50
2. Top scorer = hero creative (gets premium CPM placements)
3. Scores 2–4 = variant pool for native/banner placements
4. Below 30/50: reshoot, don't spend

## Rule
Never let one photo run everywhere. Fatigue sets in at ~3 frequency; rotate the top 3.
