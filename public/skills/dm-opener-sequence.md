# DM Opener Sequence

3-message sequence from cold follow to paid sub.

## Message 1 — on follow, within 5 minutes
Thank them + one specific personal touch. No pitch, no link.
Speed matters: <5 min response doubles reply rate.

## Message 2 — 24 hours later
Free value: a preview, a behind-the-scenes note. Still no pitch.
This message earns the right to send the third.

## Message 3 — 48 hours later
The ask, framed as an invitation with a reason:
"I only push my best stuff to subs — join today and this week's set is yours."
One link. No pressure language.

## Expectations
Cold→paid runs 3–6% on this flow when Message 1 lands fast.
If they reply at any step, drop the script and talk like a person.
