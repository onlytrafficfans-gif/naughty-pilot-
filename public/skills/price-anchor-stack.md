# Price Anchor Stack

Frame your subscription so the price feels like a steal.

## The stack (in order)
1. Name the full value first — weekly exclusive sets, DM priority, custom requests
2. Anchor high: reference what customs cost à la carte ($50+)
3. Reveal the sub price — it now reads as a discount, not a cost
4. Optional decoy: a tier priced close to the top tier pushes mid-tier picks

## Rules
- Never lead with the price
- Never apologize for the price
- Raise prices for new subs only; grandfather loyalty

## Tier template
- VIP: customs + priority DMs (anchor tier)
- Standard: full wall + weekly drops (the one most pick)
- Intro: limited wall, 1-month teaser (funnel entry)
