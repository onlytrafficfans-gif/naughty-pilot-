# A/B Test Planner

Design split tests with statistical confidence, not gut feel.

## What to test (in order of leverage)
1. Creative (biggest swing — 2–5× CTR spread)
2. Hook/first line
3. CTA frame
4. Landing page first screen
5. Price — last, and only with volume

## Rules of a real test
- One variable per test. Two changes = zero learnings.
- Pre-commit the metric (CTR? CVR? cost-per-sub?) before launch
- Minimum 1,000 impressions or 100 clicks per arm before judging
- 48h minimum — day-of-week effects are real

## Significance shortcut
If arm A doesn't beat arm B by at least 20% relative after minimums are hit, call it a tie and test something with more leverage.

## Log every test
Hypothesis → variant → result → decision. An untracked test is a coin flip you paid for.
